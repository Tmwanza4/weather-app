# Weather App

A simple weather app that shows real-time weather data for any city in the world.

🌐 **Live Demo:** https://Tmwanza4.github.io/weather-app

## Features
- Search weather by city name
- Displays temperature, weather condition, and icon
- Error handling for invalid city names
- Responsive design

## Built With
- HTML
- CSS
- JavaScript
- OpenWeatherMap API

## What I Learned
- Fetching data from a REST API using `fetch()`
- Using `async/await` to handle asynchronous code
- Dynamically updating the DOM based on API responses
- Handling errors gracefully in JavaScript
